CodeTrainer
CodeTrainer es una aplicación para practicar ejercicios de programación, seguir tu progreso y visualizar estadísticas personales en una interfaz clara y accesible.
Características
- Registro e inicio de sesión de usuarios.
- Biblioteca de ejercicios organizados por lenguaje y nivel.
- Editor integrado para escribir y probar soluciones.
- Seguimiento del progreso por ejercicio (estado, puntuación, fecha).
- Estadísticas personales y panel de progreso.
- Preferencias de usuario (tema, notificaciones)
